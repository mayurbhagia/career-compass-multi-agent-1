# Career Compass Multi-Agent System

Workshop demo for college students - Module 6: Multi-Agent AI Systems

## Architecture
- **Career Guidance Agent**: Bedrock Agent + Knowledge Base (RAG)
- **Resume Analyzer Agent**: Strands Framework + Amazon Nova Pro
- **Job Matcher Agent**: Strands + AWS MCP Tool + DynamoDB
- **Orchestrator**: Multi-Agent Coordination

## Tech Stack
- AWS Bedrock (Claude 3.5 Sonnet, Amazon Nova Pro)
- Strands Framework
- AgentCore Runtime
- DynamoDB
- Streamlit (Frontend)

## Workshop Details
- **Audience**: 650 students
- **CSAT Score**: 4.57/5
- **Based on**: AWS Bedrock Agents Workshop (Lab 2 & Lab 3)
